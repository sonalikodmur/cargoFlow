# cargoFlow
Predictive Supply Chain Disruption Detection &amp; Route Optimization System A full-stack web application that monitors shipments in real time, detects potential disruptions using rule-based risk analysis, and provides route optimization suggestions through an interactive dashboard for users and administrators.
This project presents a web-based system for supply chain monitoring and disruption detection. It enables users to track shipments, analyze risk levels based on environmental and operational factors, and receive alerts for potential issues.

The application includes role-based access, allowing administrators to manage shipments and monitor system performance through analytics dashboards. It also integrates mapping and visualization tools to enhance decision-making.

The system is built using Flask (Python) for the backend, Bootstrap for the frontend, and SQLite for database management. It serves as a scalable prototype for intelligent logistics and supply chain optimization.
